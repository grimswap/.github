| Feature | Technology | Privacy Guarantee |
|---------|------------|-------------------|
| **Sender Privacy** | LSAG Ring Signatures | Hidden among N addresses |
| **Recipient Privacy** | ERC-5564 Stealth Addresses | Unlinkable one-time address |
